Dress code

In order to maintain an enjoyable environment for all, we respectfully ask our guests to get dressed up for a great dining experience. Our Dress Code is Casual Elegant.

Athletic wear, sweatshirts, big logos, sweatpants, ball caps, excessively revealing clothing and exposed undergarments are not permitted.
